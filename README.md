# Endergonic
An open-source music visualizer, which lets you easily make homemade music videos by dynamically resizing, recoloring, rotating, and transparencizing as many images as you want.

Download both Python files together. Place them in the same directory. You must have the following nonstandard libraries:
- pillow / PIL
- numpy
- scipy
- moviepy

And these standard ones.
- tkinter
- threading
- multiprocessing
- concurrent
- roglog
- os
- sys
- json
- ctypes
